## 🐛 Bugs Fixed

### **1. Nested Expanders in Task Browser**

**Error:**
```
StreamlitAPIException: Expanders may not be nested inside other expanders.
```

**Location:** `webui/components/task_browser.py:264`

**Root Cause:**
Task Browser uses expander cards for each task, and within the "Parameters" tab, there was another expander for "View Raw Parameters". Streamlit doesn't allow nesting expanders.

**Fix:**
```python
# Before (line 264-265)
with st.expander(tr("View Raw Parameters")):
    st.json(params)

# After
st.divider()
st.caption(tr("Raw Parameters (JSON)"))
st.json(params)
```

**Impact:** Task Browser now loads without errors ✅
